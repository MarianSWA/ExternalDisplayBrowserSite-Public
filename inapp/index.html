Due to how iOS currently works, what you see on the external screen is not a native environment, but a carefully constructed "simulation" of what a desktop OS and browser would be. The app implements all OS functions from scratch, making it seem almost like a real environment. However, there are cases where limitations start to show up. This is due to security limitations, how browsers work, and decisions made by certain sites to block simulated behavior. These sites categorize this simulated behavior as bot activity (which, in this case, it clearly is not) to protect themselves and their revenue. There are also cases where limitations arise from sites using older technologies (e.g., iframes) that make simulated behavior almost impossible. For all these cases, the app offers a solution so you can get unstuck: "Switch to iPhone layout" (the gray button next to the green fullscreen button). This temporarily switches you to native iPhone mirroring so you can perform your desired action, and then return to the fully immersive experience once you're done.
